Frequently Asked Questions

Is this completely free?

Yes! We use the jsDelivr open-source network which is free for everyone. We don't charge for generating the links.

What about private repositories?

jsDelivr works best with public repositories. For private ones, you would need a token, which we are working on supporting soon.

Is there a traffic limit?

For standard use, there are no limits. jsDelivr powers huge projects like Bootstrap, so it can handle your traffic.

How do I update my cached file?

To update a file, simply push a new commit. If you use a specific version tag (recommended), create a new release. For branch links, changes may take up to 24 hours to propagate unless manually purged.

Can I use this for commercial projects?

Yes, jsDelivr is free for commercial use. Just ensure your repository settings and code licenses permit the intended usage.
